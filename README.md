# Three in Row

**Three in Row** - мобильная игра в жанре **"три в ряд"** на Java и [libGDX](https://libgdx.com/). Игрок меняет местами соседние кристаллы, собирает комбинации, выполняет цели уровней, создает бустеры и использует их для очистки поля.

Проект поддерживает desktop-запуск через LWJGL3 и сборку Android APK.

## Краткое описание

Three in Row - match-3 игра про кристаллы, уровни, бустеры и цепные реакции.

## Игровой процесс

- Игровое поле: **8x8**.
- На поле есть кристаллы шести цветов.
- Игрок меняет местами соседние кристаллы.
- Комбинация засчитывается, если после хода получается линия из **3 и более** одинаковых кристаллов.
- За очищенные кристаллы начисляются очки.
- Новые кристаллы падают сверху, при этом они появляются за пределами видимой зоны поля.
- Если возможных ходов нет, поле пересобирается без сброса прогресса уровня.

## Уровни

В игре есть система уровней с целями:

- на каждом уровне нужно собрать определенное количество кристаллов заданного цвета;
- прогресс отображается в формате `0/12`, `7/30` и т.д.;
- номер уровня сохраняется между запусками приложения;
- прогресс внутри текущего уровня не сохраняется после перезапуска;
- цели постепенно усложняются и не повторяются одним и тем же шаблоном каждые 10 уровней.

## Бустеры

В игре есть три основных типа бустеров.

### Ракета

Ракета появляется при линии из **4 кристаллов** одного цвета.

- Горизонтальная линия создает ракету, которая очищает строку.
- Вертикальная линия создает ракету, которая очищает колонку.
- При активации ракета также может запускать другие бустеры, попавшие в ее зону действия.

### Бомба

Бомба появляется при пересечении горизонтальной и вертикальной комбинации одного цвета, например при форме `T` или `L`.

- При активации бомба очищает область вокруг себя.
- Бомба может запускать другие бустеры внутри зоны взрыва.

### Молния

Молния появляется при линии из **5 и более кристаллов** одного цвета.

- При обычной активации она очищает кристаллы выбранного цвета.
- В комбинации с другими бустерами дает усиленный эффект.

## Комбинации бустеров

Бустеры можно активировать рядом друг с другом:

- ракета + ракета очищает строку и колонку;
- ракета + бомба дает усиленную очистку;
- молния + обычный кристалл очищает кристаллы цвета этого кристалла;
- молния + ракета или бомба усиливает действие по выбранному цвету;
- несколько бустеров могут запускать цепную реакцию.

## Музыка и звук

В игре есть система музыки и звуков:

- встроенные треки из `assets/music`;
- звук разрушения кристаллов;
- настройка громкости фоновой музыки;
- настройка громкости звука разрушения;
- переключение треков во время игры;
- загрузка своей музыки;
- включение и выключение отдельных треков в настройках;
- постраничный список треков в настройках.

## Интерфейс

- Адаптивный интерфейс для desktop и Android.
- Отдельные экраны главного меню, настроек и игры.
- Кнопки перезапуска, меню и переключения треков во время игры.
- Отображение счета, номера уровня, цели уровня и количества ходов.
- Визуальные эффекты для бустеров и анимации падения кристаллов.

## Структура проекта

- `core` - основная игровая логика, отрисовка, ввод, уровни и UI.
- `android` - Android-приложение, ресурсы APK и обработчик загрузки музыки.
- `lwjgl3` - desktop-запуск через LWJGL3.
- `assets` - игровые ресурсы: кристаллы, бустеры, иконки, музыка и звуки.

Основные классы:

- `Main` - главный цикл игры, состояние партии, ввод, уровни, анимации и рендеринг.
- `GameLayout` - расчет размеров и позиций UI.
- `GameAssets` - загрузка и генерация текстур.
- `GameAudio` - музыка, звуки, громкость, плейлист и переключение треков.
- `CellType` - типы клеток: обычные кристаллы, ракеты, бомбы и молнии.
- `MusicImportHandler` - общий интерфейс для загрузки пользовательской музыки.

## Ассеты

Основные ресурсы находятся в `assets`:

- `assets/gems` - текстуры кристаллов;
- `assets/rockets` - текстуры ракет;
- `assets/bombs` - текстуры бомб;
- `assets/media` - иконки интерфейса;
- `assets/music` - музыка и звук разрушения;
- `assets/screensaver.png` - изображение, используемое как иконка приложения;
- `assets/restart.png` - иконка кнопки перезапуска;
- `assets/zap.png` - текстура молнии.

## Запуск desktop-версии

Windows:

```bat
.\gradlew.bat lwjgl3:run
```

Linux/macOS:

```bash
./gradlew lwjgl3:run
```

## Проверка компиляции core

```bat
.\gradlew.bat core:compileJava
```

## Сборка Android APK

Для сборки нужен установленный Android SDK.

```bat
.\gradlew.bat :android:assembleDebug
```

Готовый debug APK будет находиться в:

```text
android/build/outputs/apk/debug/
```

## Полезные команды

```bat
.\gradlew.bat core:compileJava
```

Проверить компиляцию общего Java-кода.

```bat
.\gradlew.bat :android:assembleDebug
```

Собрать debug APK для Android.

```bat
.\gradlew.bat lwjgl3:run
```

Запустить desktop-версию.

```bat
.\gradlew.bat clean
```

Очистить build-директории.
